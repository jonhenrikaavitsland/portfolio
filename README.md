### Projects include:

- My final exam - Holidaze

## Built With:

- [React.js](https://reactjs.org/)
- [Tailwind](https://tailwindcss.com/)
- [Vite](https://vite.dev/)

## Getting Started:

### Installing

1. Clone the repo:

```bash
git clone git@github.com:jonhenrikaavitsland/portfolio.git
```

2. Install the dependencies:

```
npm install
```

### Running

```bash
npm run dev
```
